Функция inference.py служит для получения предсказания по видео, поданному на вход.
Формат видео - mp4. Длительность может быть любой.

Для работы функции неопходимо распаковать к себе на диск папку с приложением Open Face.
В нее уже загружены все необходимые веса. Ссылка для скачивания - https://disk.yandex.ru/d/3zT615U5GCbdmQ.
Там же есть массивы данных (.npy), использованные для обучения модели.

В теле функции inference.py также необходимо указать ссылку на папку, в которой будет распакован Open Face.
